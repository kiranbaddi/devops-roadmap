## What is Linux? 

Linux is not just an Operating System - it's an emotion, it is a way of Life. Sorry, I am supposed to teach you Linux not go into a monologue on how great Linux is! 
But yeah, it's the greatest Operating Sytem on Planet earth. No matter on which device you are reading this there must be linux involved somewhere in the path from the origin i.e., my computer to you. 

Now to the technical part: Linux Operating System is indeed **the** widely used operating system on Servers. Linux is a Free and Open Source Operating System. Containers and Kubernetes also run linux under the hood.

 ## Flavors of Linux

 Linux comes in different flavors. Since it's a Free and Open Source software - a lot of people have taken the codebase and added their own code on top of it and made it better and sometimes customized to some particular requirements. These flavors are known as `Distributions` or `distro` for short.

> Fun Fact: There is a distribution called Hanah Montana and even Satanic Edition Linux as well 

 ### Key distributions

There are 1000s of Linux distributions but following are the key distributions:

 - Debian: A very stable and reliable linux distribution that is popular on both personal computers and servers.

 - [Ubuntu](https://ubuntu.com/): A derivate of Debian with it's own touch. Slightly higher usage than Debian on personal computers and servers. It's maintained by [Canonical](https://canonical.com/).

 - RHEL: A very popular distribution for Enterprise servers is RedHat Enterprise Linux. It is managed by the company [RedHat Inc](https://redhat.com). RHEL is a commercial offering (paid). 

 - [Suse](https://www.suse.com/): SUSE Linux is another popular commercial choice for enterprises. It comes in 2 different licenses - open and commercial.

 - [Alpine](https://www.alpinelinux.org/): A light weight distribution that is popular in Container instances. Alpine contains only bare minimum software packages to keep the contianer images small and secure.    

## Linux != Black screen

- Unlike Windows Operating Systems servers running Linux only have a Command Line Interface (CLI) to work with. On the other hand, Windows OS even running on Servers come with a graphical user interface. Nevertheless, Linux can also be used very much as a Desktop Operating System on your personal desktops and laptops. In fact I have written this guide on my Thinkpad running [Debian](https://www.debian.org/)]. As a DevOps engineer learning Command line would make you efficient and gives you many advantages.
